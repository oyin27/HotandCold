Welcome to Hot and Cold!

Here are the rules:

- You will have 5 guesses to guess the chosen number.

- If you're more than 55 away, you are Very Cold.

- If you're within 41-55, you are Cold.

- If you're within 31-40, you are Very Cool.

- If you're within 21-30, you are Cool.

- If you're within 16-20, you are Warm.

- If you're within 9-15, you are Very Warm.

- If you're within 6-8, you are Hot.

- And if you're within 5, you are Very Hot.

If you are Very Hot, the number of your remaining guesses does not decrease.

You can not go below 1, nor can you surpass 100.

Once you've surpassed your number of guesses, you have lost.

And if you are able to guess the number, you've won!
